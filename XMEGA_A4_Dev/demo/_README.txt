These example files demonstrate the use of different peripherals
on the ATxmega32A4U Microcontroller. To build the examples follow
the instructions step by step!

1. Open Atmel Studio 7
2. File->New->Project
3. GCC C ASF Board Project
4. Select ATxmega32A4u (STK600 - ATxmega32A4U)
5. Include required modules from the ASF Wizard
6. Replace the code in src/main.c with the example code
7. Hit F7 to build the project
8. Hit F5 to upload the code and start debugging (select programmer when prompted)
